## [1.0.0] — 2026-02-18

Initial release.

### Added

- **Figma Community preset** — icon (128×128), cover image (1920×960), and screenshot frames (1920×960)
- **GitHub preset** — social preview frame (1280×640)
- **Cover template** — pre-populated with plugin name and tagline text
- **Screenshot templates** — None, Browser Mockup, Centered + Caption, Phone Mockup, Tablet Mockup
- **Image upload** — PNG/JPG images auto-resized and applied to frames; placed inside mockup placeholders or as frame fill
- **Smart re-run** — skips existing frames, only creates what's missing
- **Persistent settings** — plugin name, description, screenshot count, template, and preset saved per machine
- **Export helper** — "Select All for Export" selects all frames for File → Export
- **Dark UI** — Heavy Suite Spacegray / Base16 Ocean theme
- **Keyboard shortcuts** — Escape to cancel
- Support and feedback links
